## install

1. clone this repository first

```shell
$ git clone git@github.com:NOGUEN/so_long.git
```

2. run makefile with bonus

```shell
$ make bonus
```

3. run it with argument which has .ber extension
   (map files included)

```shell
$ ./so_long_bonus map/well_play2.ber
```
